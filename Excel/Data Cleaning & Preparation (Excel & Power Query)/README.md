## 📊 Preparing Data for Analysis (Excel & Power Query)

### 📌 Overview
This project focuses on preparing raw datasets for analysis by identifying and resolving common data quality issues. The goal was to transform multiple tables into structured datasets ready for analysis and reporting.

### 🗂 Datasets
The project involved multiple related tables:
- Customers
- Stock
- Transaction_1
- Transaction_2

### 🧹 Data Preparation Steps
- Removed duplicate records  
- Eliminated blank or unnecessary rows  
- Checked for inconsistencies and formatting issues  
- Standardized column structures  
- Merged `Transaction_1` and `Transaction_2` to create a **Master Transaction Table**

### 🛠 Tools Used
- Microsoft Excel  
- Power Query  

### 📁 Project Files
- **raw-data.xlsx** – Original datasets  
- **cleaned-data.xlsx** – Cleaned and structured datasets ready for analysis
